# Aspect Ratio

A layout primitive for preserving a fixed width-to-height ratio. It ports
shadcn's `AspectRatio` API to StyleX: pass `ratio`, put any content inside, and
customize with `sx` instead of `className`.

<Box padding="l" backgroundColor="background-card" borderRadius="m" border borderColor="border-primary">

  <AspectRatio ratio={16 / 9}>
    <Image src="https://picsum.photos/seed/aspect-ratio/960/540" alt="A landscape preview" sx={{ width: "100%", height: "100%" }} />
  </AspectRatio>

</Box>

## Usage

```tsx
import { AspectRatio } from "@/components/aspect-ratio"
```

```tsx
<AspectRatio ratio={16 / 9}>
  <Image src="/image.jpg" alt="Image" />
</AspectRatio>
```

## Examples

### Default

The default ratio is square (`1`), matching Radix's `AspectRatio.Root` default.

<Box padding="l" backgroundColor="background-card" borderRadius="m" border borderColor="border-primary">

    <Image src="https://picsum.photos/seed/aspect-square/600/600" alt="Square crop" sx={{ width: "100%", height: "100%" }} />
  </AspectRatio>

</Box>

```tsx
<AspectRatio>
  <Image src="/avatar.jpg" alt="Avatar" />
</AspectRatio>
```

### Custom Ratio

Use `ratio` for any width / height value, such as `16 / 9`, `4 / 3`, or `1`.

  </AspectRatio>

  </AspectRatio>

</Box>

```tsx
<AspectRatio ratio={16 / 9}>
  <Image src="/thumbnail.jpg" alt="Video thumbnail" />
</AspectRatio>
```

## Customization

`AspectRatio` does not accept `className` or `style`. Customize it on-system:

1. **`ratio`** ‚Äî the component's public layout decision.
2. **`sx`** ‚Äî a typed, per-instance override that accepts StyleX styles only and
   merges last.
3. **`createTheme`** ‚Äî re-skin nested content by overriding token values.

```tsx
import * as stylex from "@stylexjs/stylex"
import { borderRadius } from "@/styles/tokens.stylex"

const styles = stylex.create({
  framed: {
    overflow: "hidden",
    borderRadius: borderRadius.l,
  },
})

<AspectRatio ratio={16 / 9} sx={styles.framed}>
  <Image src="/image.jpg" alt="Image" />
</AspectRatio>
```
