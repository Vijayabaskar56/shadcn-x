# Scroll Area

A scrollable viewport built on the [Base UI](https://base-ui.com) scroll area
primitive and styled with StyleX design tokens. Behavior, overflow measurement,
thumb sizing, and scrollbar interaction come from Base UI. Appearance comes from
the shadcn-compatible `ScrollArea` and `ScrollBar` slots, with `sx` replacing
`className`.

## Usage

```tsx
import { ScrollArea, ScrollBar } from "@/components/scroll-area"
```

```tsx
<ScrollArea>
  <Text variant="p">Scrollable content</Text>
</ScrollArea>
```

## Customization

`ScrollArea` and `ScrollBar` never accept `className`. Customize them on-system:

1. **`sx`** â€” a typed, per-instance StyleX override, merged last.
2. **`createTheme`** â€” re-skin a subtree by overriding token values.
3. **Owned source** â€” edit the copied component when you need a structural
   scrollbar change.

```tsx
import * as stylex from "@stylexjs/stylex"

const styles = stylex.create({
  compact: {
    width: "16rem",
    height: "10rem",
  },
})

<ScrollArea sx={styles.compact}>
  <Text variant="p">Scrollable content</Text>
</ScrollArea>
```
