# Avatar

`Avatar` wraps Base UI's avatar behavior with shadcn's slot API and shadcn-x
StyleX tokens. It supports `Avatar`, `AvatarImage`, `AvatarFallback`,
`AvatarGroup`, `AvatarGroupCount`, and `AvatarBadge`.

<Box display="flex" alignItems="center" gap="m" padding="l" backgroundColor="background-card" borderRadius="m" border borderColor="border-primary">
  <Avatar>
    <AvatarImage src="/avatar.jpg" alt="Vijay" />
    <AvatarFallback>VB</AvatarFallback>
    <AvatarBadge />
  </Avatar>
  <Avatar>
    <AvatarFallback>SC</AvatarFallback>
  </Avatar>
</Box>

## Usage

```tsx
import {
  Avatar,
  AvatarBadge,
  AvatarFallback,
  AvatarGroup,
  AvatarGroupCount,
  AvatarImage,
} from "@/components/avatar"
```

```tsx
<Avatar>
  <AvatarImage src="/avatar.jpg" alt="Vijay" />
  <AvatarFallback>VB</AvatarFallback>
</Avatar>
```

## Customization

Every public slot accepts `sx`, a typed StyleX style merged last. Use tokens for
theme decisions and keep broader changes in owned source.

```tsx
import * as stylex from "@stylexjs/stylex"
import { colors } from "@/styles/tokens.stylex"

const styles = stylex.create({
  status: {
    backgroundColor: colors.success,
  },
})

<Avatar>
  <AvatarFallback>VB</AvatarFallback>
  <AvatarBadge sx={styles.status} />
</Avatar>
```

## API Reference

| Component            | Prop   | Type                                | Default     |
| -------------------- | ------ | ----------------------------------- | ----------- |
| `Avatar`             | `size` | `"default" \| "sm" \| "lg"`         | `"default"` |
| `Avatar`             | `sx`   | `StyleXStyles`                      | —           |
| `AvatarImage`        | `sx`   | `StyleXStyles`                      | —           |
| `AvatarFallback`     | `delay`| `number`                            | —           |
| `AvatarFallback`     | `sx`   | `StyleXStyles`                      | —           |
| `AvatarBadge`        | `sx`   | `StyleXStyles`                      | —           |
| `AvatarGroup`        | `sx`   | `StyleXStyles`                      | —           |
| `AvatarGroupCount`   | `sx`   | `StyleXStyles`                      | —           |

`Avatar`, `AvatarImage`, and `AvatarFallback` forward Base UI avatar props
except `className`, `style`, and `color`. The group, count, and badge slots
forward standard host props except `className`, `style`, and `color`.
