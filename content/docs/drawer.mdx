# Drawer

A drawer built on the [vaul](https://vaul.emilkowalski.dev) primitive and styled
with StyleX design tokens. Behavior and accessibility — drag-to-dismiss, focus
management, scroll lock, portal stacking — come from vaul; appearance comes from
a closed set of typed `sx` props. Never accepts `className`.

## Usage

```tsx
import {
  Drawer,
  DrawerClose,
  DrawerContent,
  DrawerDescription,
  DrawerFooter,
  DrawerHeader,
  DrawerTitle,
  DrawerTrigger,
} from "@/components/drawer"
```

```tsx
<Drawer>
  <DrawerTrigger render={<Button variant="outline">Open Drawer</Button>} />
  <DrawerContent>
    <DrawerHeader>
      <DrawerTitle>Move Goal</DrawerTitle>
      <DrawerDescription>Set your daily activity goal.</DrawerDescription>
    </DrawerHeader>
    <DrawerFooter>
      <Button>Confirm</Button>
      <DrawerClose render={<Button variant="outline">Cancel</Button>} />
    </DrawerFooter>
  </DrawerContent>
</Drawer>
```

> **Trigger as another element.** `DrawerTrigger` renders a `<button>` by
> default. To style another element as the trigger (e.g. a `Button`), pass it
> via the `render` prop (or vaul's `asChild`) — never `className`.

drawers. Set `showHandle={false}` and place your own, or drop a `DrawerHandle`

## Customization

`Drawer` never accepts `className`. Customize it on-system, in order of reach:

1. **`sx`** — a typed, per-instance override on any subcomponent that accepts
   `StyleXStyles` and merges last, so it wins. Useful for width or spacing
   tweaks on `DrawerContent` / `DrawerHeader` / `DrawerFooter`.
2. **`createTheme`** — re-skin a whole subtree by overriding token *values*
   (e.g. `colors`, `borderRadius`, `boxShadow`).

```tsx
import * as stylex from "@stylexjs/stylex"

const styles = stylex.create({
  wide: { width: "100%" },
})

<Drawer direction="right">
  <DrawerTrigger render={<Button variant="outline">Wide</Button>} />
  <DrawerContent sx={styles.wide}>
    <DrawerHeader>
      <DrawerTitle>Wider content</DrawerTitle>
      <DrawerDescription>
        The default max width is overridden via the last-wins sx prop.
      </DrawerDescription>
    </DrawerHeader>
  </DrawerContent>
</Drawer>
```

## API Reference

`Drawer` (Root) accepts all [vaul Drawer Root
props](https://vaul.emilkowalski.dev) (`open`, `onOpenChange`, `defaultOpen`,
`direction`, `dismissible`, `shouldScaleBackground`, `snapPoints`, etc.).

| Subcomponent        | Prop          | Type                                  | Default   |
| ------------------- | ------------- | ------------------------------------- | --------- |
| `Drawer`            | `direction`   | `"top" \| "bottom" \| "left" \| "right"` | `"bottom"` |
| `DrawerTrigger`     | `render`      | element                               | —         |
|                     | `sx`          | `StyleXStyles`                        | —         |
| `DrawerContent`     | `showHandle`  | `boolean`                             | `true`    |
|                     | `sx`          | `StyleXStyles`                        | —         |
| `DrawerHandle`      | `sx`          | `StyleXStyles`                        | —         |
| `DrawerHeader`      | `sx`          | `StyleXStyles`                        | —         |
| `DrawerFooter`      | `sx`          | `StyleXStyles`                        | —         |
| `DrawerTitle`       | `sx`          | `StyleXStyles`                        | —         |
| `DrawerDescription` | `sx`          | `StyleXStyles`                        | —         |
| `DrawerClose`       | `render`      | element                               | —         |
|                     | `sx`          | `StyleXStyles`                        | —         |
| `DrawerOverlay`     | `sx`          | `StyleXStyles`                        | —         |
| `DrawerPortal`      | —             | vaul Portal                           | —         |

All other props (native attributes, `render`, vaul behavior props) are forwarded
to the underlying vaul primitive.
