# Context Menu

A context menu built on the [Base UI](https://base-ui.com) context menu
primitive and styled with StyleX design tokens. Behavior and accessibility ŌĆö
right-click trigger, positioning, roving focus, keyboard navigation, submenu, and
portal stacking ŌĆö come from Base UI; appearance comes from a closed set of typed
`inset` / `variant` / `sx` props. Never accepts `className`.

**Right-click the area below to open the menu.**

## Usage

```tsx

  ContextMenu,
  ContextMenuContent,
  ContextMenuItem,
  ContextMenuShortcut,
  ContextMenuTrigger,
} from "@/components/context-menu"
```

```tsx

  <ContextMenuTrigger>Right-click me</ContextMenuTrigger>
  <ContextMenuContent>
    <ContextMenuItem>
      Back
      <ContextMenuShortcut>Ōīś[</ContextMenuShortcut>
    </ContextMenuItem>
    <ContextMenuItem>Forward</ContextMenuItem>
  </ContextMenuContent>
</ContextMenu>
```

### Checkbox items

Toggle settings on or off with `ContextMenuCheckboxItem`. The check indicator
appears automatically when the item is checked.

<Box padding="l" backgroundColor="background-card" borderRadius="m" border borderColor="border-primary">

<ContextMenu>
  <ContextMenuTrigger>
    <Box padding="2xl" backgroundColor="background-muted" borderRadius="m" textAlign="center" color="text-secondary">
      Right-click here
    </Box>
  </ContextMenuTrigger>
  <ContextMenuContent>
    <ContextMenuCheckboxItem checked>Show status bar</ContextMenuCheckboxItem>
    <ContextMenuCheckboxItem>Show side panel</ContextMenuCheckboxItem>
  </ContextMenuContent>
</ContextMenu>

</Box>

```tsx
function SettingsMenu() {
  const [statusBar, setStatusBar] = React.useState(true)
  return (
    <ContextMenu>
      <ContextMenuTrigger>Right-click me</ContextMenuTrigger>
      <ContextMenuContent>
        <ContextMenuCheckboxItem
          checked={statusBar}
          onCheckedChange={setStatusBar}
        >
          Show status bar
        </ContextMenuCheckboxItem>
      </ContextMenuContent>
    </ContextMenu>
  )
}
```

## Customization

`ContextMenu` never accepts `className`. Customize it on-system, in order of
reach:

1. **`sx`** ŌĆö a typed, per-instance override on any subcomponent that accepts
   `StyleXStyles` and merges last, so it wins. Useful for width, alignment, or
   spacing tweaks on `ContextMenuContent` / `ContextMenuItem`.
2. **`createTheme`** ŌĆö re-skin a whole subtree by overriding token *values*
   (e.g. `colors`, `borderRadius`, `boxShadow`).

```tsx
import * as stylex from "@stylexjs/stylex"

const styles = stylex.create({
  wide: { minWidth: "16rem" },
})

<ContextMenu>
  <ContextMenuTrigger>Right-click me</ContextMenuTrigger>
  <ContextMenuContent sx={styles.wide}>
    <ContextMenuItem>Open</ContextMenuItem>
  </ContextMenuContent>
</ContextMenu>
```

## API Reference

`ContextMenu` (aliased from `ContextMenuPrimitive.Root`) accepts all [Base UI
Context Menu Root props](https://base-ui.com/react/components/context-menu)
(`open`, `onOpenChange`, etc.).

| ------------------------- | -------------- | --------------------------------------------------- | ----------- |
| `ContextMenuTrigger`      | `sx`           | `StyleXStyles`                                      | ŌĆö           |
| `ContextMenuContent`      | `side`         | `"top" \| "bottom" \| "left" \| "right" \| ŌĆ”`       | `"right"`   |
|                           | `sideOffset`   | `number`                                            | `0`         |
|                           | `align`        | `"start" \| "center" \| "end"`                      | `"start"`   |
|                           | `alignOffset`  | `number`                                            | `4`         |
|                           | `sx`           | `StyleXStyles`                                      | ŌĆö           |
| `ContextMenuGroup`        | `sx`           | `StyleXStyles`                                      | ŌĆö           |
| `ContextMenuLabel`        | `inset`        | `boolean`                                           | `false`     |
|                           | `sx`           | `StyleXStyles`                                      | ŌĆö           |
| `ContextMenuItem`         | `inset`        | `boolean`                                           | `false`     |
|                           | `variant`      | `"default" \| "destructive"`                        | `"default"` |
|                           | `sx`           | `StyleXStyles`                                      | ŌĆö           |
| `ContextMenuCheckboxItem` | `checked`      | `boolean`                                           | ŌĆö           |
|                           | `inset`        | `boolean`                                           | `false`     |
|                           | `sx`           | `StyleXStyles`                                      | ŌĆö           |
| `ContextMenuRadioGroup`   | `value`        | `any`                                               | ŌĆö           |
|                           | `sx`           | `StyleXStyles`                                      | ŌĆö           |
| `ContextMenuRadioItem`    | `value`        | `any`                                               | ŌĆö           |
