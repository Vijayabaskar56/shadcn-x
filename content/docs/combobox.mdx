# Combobox

An autocomplete combobox built on the [Base UI](https://base-ui.com) combobox primitive
and styled with StyleX design tokens. Behavior and accessibility come from Base
UI; appearance comes from typed `sx` props.

## Examples

### With filtering

Base UI provides `useFilter` and `useFilteredItems` for client-side filtering.

```tsx
import { useFilter, useFilteredItems } from "@base-ui/react/combobox"

function FilteredCombobox() {
  const allItems = [
    { value: "us", label: "United States" },
    { value: "ca", label: "Canada" },
    { value: "mx", label: "Mexico" },
  ]

  return (
    <Combobox>
      <ComboboxInput placeholder="Search..." />
      <ComboboxContent>
        <ComboboxList>
          {allItems.map((item) => (
            <ComboboxItem key={item.value} value={item.value}>
              {item.label}
            </ComboboxItem>
          ))}
        </ComboboxList>
        <ComboboxEmpty>No results found.</ComboboxEmpty>
      </ComboboxContent>
    </Combobox>
  )
}
```

## Customization

Combobox never accepts `className`. Customize it on-system, in order of reach:

1. **`sx`** — a typed, per-instance override on any subcomponent that accepts
   `StyleXStyles` and merges last.
2. **`createTheme`** — re-skin a whole subtree by overriding token *values*.

```tsx
import * as stylex from "@stylexjs/stylex"

const styles = stylex.create({
  wide: { width: "100%" },
})

<Combobox>
  <ComboboxTrigger aria-label="Full" sx={styles.wide}>
    <ComboboxValue placeholder="Full width" />
  </ComboboxTrigger>
  <ComboboxContent>
    <ComboboxItem value="a">Option A</ComboboxItem>
  </ComboboxContent>
</Combobox>
```

## API Reference

`Combobox` (aliased from `ComboboxPrimitive.Root`) accepts all [Base UI Combobox
Root props](https://base-ui.com/react/components/combobox).

| Subcomponent            | Prop              | Type                                    | Default     |
| ----------------------- | ----------------- | --------------------------------------- | ----------- |
| `ComboboxTrigger`       | `sx`              | `StyleXStyles`                          | —           |
| `ComboboxValue`         | `sx`              | `StyleXStyles`                          | —           |
| `ComboboxClear`         | `sx`              | `StyleXStyles`                          | —           |
| `ComboboxInput`         | `showTrigger`     | `boolean`                               | `true`      |
|                         | `showClear`       | `boolean`                               | `false`     |
|                         | `sx`              | `StyleXStyles`                          | —           |
| `ComboboxContent`       | `side`            | `"top" \| "bottom"`                     | `"bottom"`  |
|                         | `sideOffset`      | `number`                                | `4`         |
|                         | `align`           | `"start" \| "center" \| "end"`          | `"start"`   |
|                         | `alignOffset`     | `number`                                | `0`         |
|                         | `anchor`          | `React.RefObject<HTMLElement \| null>`  | —           |
|                         | `sx`              | `StyleXStyles`                          | —           |
| `ComboboxList`          | `sx`              | `StyleXStyles`                          | —           |
| `ComboboxItem`          | `sx`              | `StyleXStyles`                          | —           |
| `ComboboxGroup`         | `sx`              | `StyleXStyles`                          | —           |
| `ComboboxLabel`         | `sx`              | `StyleXStyles`                          | —           |
| `ComboboxCollection`    | `sx`              | `StyleXStyles`                          | —           |
| `ComboboxEmpty`         | `sx`              | `StyleXStyles`                          | —           |
| `ComboboxSeparator`     | `sx`              | `StyleXStyles`                          | —           |
| `ComboboxChips`         | `sx`              | `StyleXStyles`                          | —           |
| `ComboboxChip`          | `showRemove`      | `boolean`                               | `true`      |
|                         | `sx`              | `StyleXStyles`                          | —           |
| `ComboboxChipsInput`    | `sx`              | `StyleXStyles`                          | —           |

All other props (native attributes, `render`, etc.) are forwarded to the
underlying Base UI primitive.
