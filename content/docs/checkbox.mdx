# Checkbox

A checkbox built on the [Base UI](https://base-ui.com/react/components/checkbox) `Checkbox` primitive and styled with StyleX design tokens. Behavior and accessibility (including `role="checkbox"`, keyboard navigation, and ARIA attributes) come from Base UI; appearance maps 1:1 to shadcn's checkbox.

<Box display="flex" alignItems="center" gap="m" padding="l" backgroundColor="background-card" borderRadius="m" border borderColor="border-primary">
  <Checkbox id="terms" />
  <Label htmlFor="terms">Accept terms and conditions</Label>
</Box>

## Usage

```tsx
import { Checkbox } from "@/components/checkbox"
```

```tsx
<Checkbox defaultChecked />
```

## Controlled

<Box padding="l" backgroundColor="background-card" borderRadius="m" border borderColor="border-primary">
  <Checkbox checked readOnly />
</Box>

```tsx
const [checked, setChecked] = React.useState(false)

<Checkbox
  checked={checked}
  onCheckedChange={(v) => setChecked(v)}
/>
```

## Customization

`Checkbox` accepts no `className`. Customize it on-system:

1. **`sx`** ã a typed, per-instance override (StyleX styles only), merged last.
2. **`createTheme`** ã re-skin a subtree by overriding token *values*.

```tsx
import * as stylex from "@stylexjs/stylex"

const styles = stylex.create({
  large: { width: "1.5rem", height: "1.5rem" },
})

<Checkbox sx={styles.large} />
```
