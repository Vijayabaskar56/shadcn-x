# InputGroup

A set of components for composing `Input` or `Textarea` with surrounding addons,
buttons, and text — commonly used for search bars, currency fields, icon
prefixes, and compound form controls.

All components use existing shadcn-x primitives (`Input`, `Textarea`, `Button`)
under the hood.

## Subcomponents

| Component              | Delegates to | Slot                            |
| ---------------------- | ------------ | ------------------------------- |
| `InputGroup`           | `div`        | `input-group`                   |
| `InputGroupAddon`      | `div`        | `input-group-addon`             |
| `InputGroupButton`     | `Button`     | `button` (inherited)            |
| `InputGroupText`       | `span`       | `input-group-text`              |
| `InputGroupInput`      | `Input`      | `input-group-control`           |
| `InputGroupTextarea`   | `Textarea`   | `input-group-control`           |

## Usage

```tsx
import {
  InputGroup,
  InputGroupAddon,
  InputGroupButton,
  InputGroupInput,
  InputGroupText,
} from "@/components/input-group"
```

## API Reference

### InputGroup

| Prop | Type         | Default |
| ---- | ------------ | ------- |
| `sx` | `StyleXStyles` | —       |

All forwarded `<div>` props accepted.

### InputGroupAddon

| Prop    | Type                                                             | Default          |
| ------- | ---------------------------------------------------------------- | ---------------- |
| `align` | `"inline-start" \| "inline-end" \| "block-start" \| "block-end"` | `"inline-start"` |
| `sx`    | `StyleXStyles`                                                     | —                |

### InputGroupButton

| Prop      | Type                               | Default   |
| --------- | ---------------------------------- | --------- |
| `type`    | `"button" \| "submit" \| "reset"`  | `"button"` |
| `variant` | Button variants                    | `"ghost"` |
| `size`    | `"xs" \| "sm" \| "icon-xs" \| "icon-sm"` | `"xs"`   |
| `sx`      | `StyleXStyles`                     | —         |

### InputGroupInput, InputGroupTextarea

Forward all props from `Input` / `Textarea` respectively, plus `sx`.
