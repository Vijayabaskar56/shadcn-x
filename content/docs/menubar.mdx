# Menubar

A menubar built on the [Base UI](https://base-ui.com) menubar and menu
primitives and styled with StyleX design tokens. Behavior and accessibility — a
horizontal bar of triggers with roving focus, dropdown positioning, keyboard
navigation, submenu, and portal stacking — come from Base UI; appearance comes
from a closed set of typed `inset` / `variant` / `sx` props. Never accepts
`className`.

## Usage

```tsx
import {
  Menubar,
  MenubarContent,
  MenubarItem,
  MenubarMenu,
  MenubarShortcut,
  MenubarTrigger,
} from "@/components/menubar"
```

```tsx
<Menubar>
  <MenubarMenu>
    <MenubarTrigger>File</MenubarTrigger>
    <MenubarContent>
      <MenubarItem>
        New File
        <MenubarShortcut>⌘N</MenubarShortcut>
      </MenubarItem>
      <MenubarItem>Open…</MenubarItem>
    </MenubarContent>
  </MenubarMenu>
</Menubar>
```

### Checkbox item

Toggle a setting on or off with `MenubarCheckboxItem`. The check indicator
appears automatically when the item is checked.

<Menubar>

<MenubarMenu>
  <MenubarTrigger>View</MenubarTrigger>
  <MenubarContent>
    <MenubarCheckboxItem checked>Show status bar</MenubarCheckboxItem>
    <MenubarCheckboxItem>Show side panel</MenubarCheckboxItem>
  </MenubarContent>
</MenubarMenu>

</Menubar>

```tsx
function ViewMenu() {
  const [statusBar, setStatusBar] = React.useState(true)
  return (
    <Menubar>
      <MenubarMenu>
        <MenubarTrigger>View</MenubarTrigger>
        <MenubarContent>
          <MenubarCheckboxItem
            checked={statusBar}
            onCheckedChange={setStatusBar}
          >
            Show status bar
          </MenubarCheckboxItem>
        </MenubarContent>
      </MenubarMenu>
    </Menubar>
  )
}
```

## Customization

`Menubar` never accepts `className`. Customize it on-system, in order of reach:

1. **`sx`** — a typed, per-instance override on any subcomponent that accepts
   `StyleXStyles` and merges last, so it wins. Useful for width, alignment, or
   spacing tweaks on `Menubar` / `MenubarContent` / `MenubarItem`.
2. **`createTheme`** — re-skin a whole subtree by overriding token *values*
   (e.g. `colors`, `borderRadius`, `boxShadow`).

```tsx
import * as stylex from "@stylexjs/stylex"

const styles = stylex.create({
  wide: { minWidth: "16rem" },
})

<Menubar>
  <MenubarMenu>
    <MenubarTrigger>File</MenubarTrigger>
    <MenubarContent sx={styles.wide}>
      <MenubarItem>Open</MenubarItem>
    </MenubarContent>
  </MenubarMenu>
</Menubar>
```

## API Reference

`Menubar` accepts all [Base UI Menubar Root props](https://base-ui.com/react/components/menubar)
(`modal`, `orientation`, `loopFocus`, etc.). Each `MenubarMenu` maps to a
[Base UI Menu Root](https://base-ui.com/react/components/menu) (`open`,
`onOpenChange`, etc.).

| Subcomponent             | Prop           | Type                                                | Default     |
| ------------------------ | -------------- | --------------------------------------------------- | ----------- |
| `Menubar`                | `sx`           | `StyleXStyles`                                      | —           |
| `MenubarMenu`            | —              | Base UI Menu Root                                   | —           |
| `MenubarTrigger`         | `sx`           | `StyleXStyles`                                      | —           |
| `MenubarContent`         | `side`         | `"top" \| "bottom" \| "left" \| "right" \| …`       | `"bottom"`  |
|                          | `sideOffset`   | `number`                                            | `8`         |
|                          | `align`        | `"start" \| "center" \| "end"`                      | `"start"`   |
|                          | `alignOffset`  | `number`                                            | `-4`        |
|                          | `sx`           | `StyleXStyles`                                      | —           |
| `MenubarGroup`           | `sx`           | `StyleXStyles`                                      | —           |
| `MenubarLabel`           | `inset`        | `boolean`                                           | `false`     |
|                          | `sx`           | `StyleXStyles`                                      | —           |
| `MenubarItem`            | `inset`        | `boolean`                                           | `false`     |
|                          | `variant`      | `"default" \| "destructive"`                        | `"default"` |
|                          | `sx`           | `StyleXStyles`                                      | —           |
| `MenubarCheckboxItem`    | `checked`      | `boolean`                                           | —           |
|                          | `inset`        | `boolean`                                           | `false`     |
|                          | `sx`           | `StyleXStyles`                                      | —           |
| `MenubarRadioGroup`      | `value`        | `any`                                               | —           |
|                          | `sx`           | `StyleXStyles`                                      | —           |
| `MenubarRadioItem`       | `value`        | `any`                                               | —           |
|                          | `inset`        | `boolean`                                           | `false`     |
|                          | `sx`           | `StyleXStyles`                                      | —           |
| `MenubarSub`             | —              | Base UI SubmenuRoot                                 | —           |
| `MenubarSubTrigger`      | `inset`        | `boolean`                                           | `false`     |
|                          | `sx`           | `StyleXStyles`                                      | —           |
| `MenubarSubContent`      | `side`         | `"top" \| "bottom" \| "left" \| "right" \| …`       | `"right"`   |
|                          | `sx`           | `StyleXStyles`                                      | —           |
| `MenubarSeparator`       | `sx`           | `StyleXStyles`                                      | —           |
| `MenubarShortcut`        | `sx`           | `StyleXStyles`                                      | —           |
| `MenubarPortal`          | —              | Base UI Portal                                      | —           |

All other props (native attributes, `render`, etc.) are forwarded to the
underlying Base UI primitive.
